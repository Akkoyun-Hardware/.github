# PCB Technical References

Central PCB reference library for the Akkoyun-Hardware organization.

## Contents

- `Footprints/` — package size reference images (`0201`, `0402`, `0603`, `0805`, `1206`, `2512`)
- `Stackups/` — layer-count / thickness stack-up visuals and reference captures
- `Stackups/PDF/` — manufacturer datasheets, design guides, and HDI / material references

## Purpose

These documents are shared technical references for hardware design work across the organization.
They are intentionally stored in the organization repository instead of a single product/module repository.

## Notes

- Product-specific documents should live in the related hardware repository.
- Cross-project PCB process knowledge and stack-up references should live here.
